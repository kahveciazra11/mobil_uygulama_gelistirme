# mobil_uygulama_gelistirme
# Pembe Temalı Mini Katalog Ödevi

##  Açıklama
Bu projede, Flutter kullanarak pembe temalı küçük bir ürün kataloğu geliştirdim. Uygulamada ürünlerin listelendiği bir ana sayfa, ürün detay ekranı ve alt menü (BottomNavigationBar) ile geçiş yapılabilen bir sepet sayfası bulunmaktadır. Proje, yönergeye uygun olarak hiçbir ekstra paket kullanılmadan, yalnızca varsayılan `material.dart` paketi ile geliştirilmiştir.

## Geliştirme Ortamı ve Sürümler
Eğitim yönergesindeki zorunlu araçlar kullanılarak ortam hazırlanmıştır:
- **Flutter SDK:** 3.41.9
- **Dart SDK:** 3.11.5
- **Geliştirici Ortamı (IDE):** Visual Studio Code
- **Test Ortamı:** Android Studio (Android Emulator kullanılarak test edilmiştir)

## Çalıştırma Adımları
1. Projeyi GitHub üzerinden bilgisayarınıza indirin (.zip olarak veya `git clone` ile).
2. İndirdiğiniz proje klasörünü **Visual Studio Code** ile açın.
3. VS Code terminalini açarak `flutter pub get` komutunu çalıştırın.
4. Arka planda Android Studio üzerinden bir **Android Emulator** başlatın (veya fiziksel Android cihazınızı bağlayın).
5. VS Code terminalinde `flutter run` komutunu çalıştırarak uygulamayı cihazınızda test edin.
